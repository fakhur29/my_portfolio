# 📂 Featured Projects

### 🧠 Intelligent Data Analysis System
An AutoML platform that automates the entire data analysis pipeline—including data preprocessing, dynamic EDA, model training, and predictions—complete with model explainability through feature importance and confusion matrices in a single click.
* **Technologies:** Python, Pandas, NumPy, Scikit-Learn, Streamlit, Seaborn

---

### 📊 Interactive Data Visualization Web App
A modern, custom-styled web application allowing users to upload any dataset and instantly generate real-time dynamic plots, including scatter, bar, histogram, box, pair, and facet grid charts.
* **Technologies:** Python, Pandas, Seaborn, Streamlit, CSS

---

### 📚 AI-Powered Book Recommendation System
A smart recommendation engine that suggests similar books based on a user-selected title, built using K-Nearest Neighbors (KNN) with cosine similarity and optimized with Truncated SVD for handling high-dimensional sparse data.
* **Technologies:** Python, Pandas, NumPy, Scikit-Learn, Streamlit
